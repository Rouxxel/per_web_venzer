# Portfolio frontend

Frontend for the personal portfolio site (see repo root `idea.md`). Single-page app: Hero, About, Projects, Experience, Contact. Built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and shadcn/ui. Deployable to Vercel.

- **`src/sections/`** — Page sections (Hero, About, Projects, Experience, Contact)
- **`src/components/`** — Shared UI (Navbar, Footer, NavLink) and `ui/` (shadcn)
- **`src/data/`** — Static content (projects, experience, skills)
- **`public/images/`** — Portfolio images (profile, project screenshots)

```bash
npm install
npm run dev
npm run build
```
